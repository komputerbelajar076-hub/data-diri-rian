Jenis Selector CSS

  • element { } → selector berdasarkan nama elemen.
    Contoh: p { color: blue; }
  • .class { } → selector berdasarkan nama kelas.
    Contoh: .judul { font-size: 24px; }
  • #id { } → selector berdasarkan ID unik.
    Contoh: #utama { background: yellow; }
  • * { } → universal selector (semua elemen).
    Contoh: * { margin: 0; }
  • element element { } → descendant selector.
    Contoh: div p { color: red; }
  • element:hover { } → pseudo-class selector.
    Contoh: a:hover { color: green; }
